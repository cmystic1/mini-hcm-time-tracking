# Mini HCM Time Tracking

A lightweight Human Capital Management (HCM) Time Tracking System built as a technical assessment.

## Tech Stack

- React (Vite)
- Node.js
- Express
- Firebase Authentication
- Cloud Firestore

## Features

- User Registration
- User Authentication
- Punch In / Punch Out
- Attendance Tracking
- Daily Summary
- Admin Dashboard
