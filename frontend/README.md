# Mini HCM Time Tracking

## Features

- User Registration
- User Login
- Punch In
- Punch Out
- Attendance History
- Attendance Summary
- Firebase Authentication
- Firestore Database

## Tech Stack

Frontend
- React
- Vite
- Bootstrap
- Firebase Authentication

Backend
- Node.js
- Express.js
- Firebase Admin SDK
- Firestore

## Installation

Frontend

npm install
npm run dev

Backend

npm install
npm run dev
