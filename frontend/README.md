# Mini HCM Time Tracking

## Features

- Employee Registration
- Employee Login
- Punch In
- Punch Out
- Attendance History
- Attendance Summary

## Tech Stack

Frontend
- React
- Bootstrap
- Firebase Authentication

Backend
- Node.js
- Express.js
- Firebase Admin SDK
- Firestore

## Installation

### Backend

npm install
npm run dev

### Frontend

npm install
npm run dev
